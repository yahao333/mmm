# 项目概览
这是一个用于监控 MinMax 的账号信息的前端工具，目前只有一个功能： 在 https://platform.minimaxi.com/user-center/payment/coding-plan 页面上查找当前使用量，如果超过90%就进行预警

这是一个生产环境的前端项目。
目标是：正确性、清晰度和可维护性。

# 技术栈

- Tauri 跨平台桌面应用
- React
- TypeScript（严格模式）
- Tailwind CSS

# 规则

- 不得引入新库
- 不得重构无关代码
- 优先选择显式、易读的代码，而非抽象写法

# 预期

- 如有不确定之处，请在编码前说明假设条件
- 遵循代码库中现有的模式

# 一般规则
- 前端项目使用 pnpm 进行包管理
- 详细编写中文代码注释
- 编写详细的调试日志
- rust 使用日志库打印日志
